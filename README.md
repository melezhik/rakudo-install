# SYNOPSIS

Install Rakudo as system package.

# Description

This is simple installer of Rakudo Perl6.

Platforms supported:

* CentOS7
* Debian

# Install

    $ sparrow plg install rakudo-install

# Install

    $ sparrow plg run rakudo-install
